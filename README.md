# 🛒 ShopEasy - Shopping Cart Application
ShopEasy - Professional Shopping Cart Application: A fully functional, responsive e-commerce shopping cart application with admin panel, built with vanilla HTML, CSS, and JavaScript. Features include product management, shopping cart functionality, and persistent storage using localStorage.

## 🚀 Live Demo

**View Live App:** (https://aqsaalirazajamali.github.io/shoppingCard-html-project/)

## 📋 About The Project

A fully functional, professional e-commerce shopping cart application built with vanilla HTML, CSS, and JavaScript. Perfect for small businesses or as a learning resource.

### ✨ Key Features

**For Customers:**
- 🛍️ Browse products in beautiful grid layout
- 🛒 Add/remove items from shopping cart
- 🔢 Update product quantities
- 💰 Real-time price calculation with tax & shipping
- 💾 Cart saves automatically (localStorage)

**For Admins:**
- ➕ Add new products (name, price, image)
- ❌ Delete existing products
- 📊 Visual product counter (max 20 products)
- 🔄 Instant updates across all pages

## 🛠️ Built With

- **HTML5** - Structure
- **CSS3** - Styling & Animations
- **JavaScript (ES6)** - Functionality
- **Font Awesome** - Icons
- **localStorage API** - Data Persistence
